Testing Frameworks: 

JUnit - 
Regression Testing 
Integration Testing
Commonly used
Runs in command line or modern Java IDE
Open Source

Serenity- 
Extensively used
Open Source
Behaviour Driven Development 
Attuned with Selenium WebDriver

TestNG 
Derived from JUnit
Supports Integration testing
Open Source
Supports advanced notation
Tests can run in parallel
Easier than JUnit

Recommendation: 

To use TestNG however team members familiarity with JUnit may be more feasible as time constraints do not allow for learning new frameworks.
